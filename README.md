### Steps
1. Copy `.env` file:
    ```bash
    cp .env_example .env
    ```

2. Build:
    ```bash
    make build
    ```

3. Start:
    ```bash
    make start
    ```

4. Stop:
    ```bash
    make stop
    ```
# Other `make` commands
- Restart:
    ```bash
    make restart
    ```

- Remove:
    ```bash
    make remove
    ```
